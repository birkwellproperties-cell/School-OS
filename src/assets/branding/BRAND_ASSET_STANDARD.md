# SchoolOS Brand Asset Naming Standard

## Product Name

The official product name is:

SchoolOS

Use the exact capitalization above in all visible product copy, metadata,
accessibility labels, documentation, and social-sharing content.

## File Naming Rules

- Use lowercase kebab-case.
- Prefix product-specific assets with `schoolos-`.
- Include the asset type or lockup orientation.
- Include the visual variant when relevant.
- SVG is the canonical format for logos and marks.
- PNG is used only where a platform requires raster artwork.

## Usage Rules

- Use the primary logo on neutral or light backgrounds.
- Use the light logo on dark backgrounds.
- Use the dark logo where monochrome dark artwork is required.
- Use the mark-only asset for favicons, compact navigation, launch icons,
  and constrained square surfaces.
- Do not stretch, rotate, recolor, outline, or independently rearrange
  logo elements.
- Preserve the SVG viewBox and aspect ratio.
